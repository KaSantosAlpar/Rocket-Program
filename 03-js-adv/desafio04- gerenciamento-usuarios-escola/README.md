Desafio 4 
JS Adv. 

Vamos iniciar a construção de um app AngularJS para gerenciar usuários (alunos e professores). 

Objetivo: 
Exibir uma mensagem simples na tela utilizando os principais recursos iniciais do 
AngularJS. 

Requisitos: 
Crie uma estrutura HTML com: 
-> Tag ng-app para iniciar a aplicação 
-> Um ng-controller chamado AppController 
-> No controller, crie uma variável mensagem com o valor: 

"Bem-vindo ao sistema de cadastro escolar" 

-> Use ng-bind para exibir essa mensagem na tela. 
-> Adicione mais uma variável usuario = { nome: "João", tipo: "Aluno" } e mostre os dados do usuário na tela com {{ }} ou ng-bind. 
Mostre uma imagem condicional com ng-if, dependendo se o tipo do usuário é "Aluno" ou "Professor" 

Guarde esse desafio no repositório de sempre na pasta: gerenciamento-usuarios-escola
