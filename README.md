Temos que pegar! Temos, mas o que?
Rocket Sul,
 
Nos idos dos anos 1990 fomos (eu fui, vocês não rs) pegos pela febre Pokémon! Bichinhos lutando em arenas para conseguir insígnias e se tornar o maior mestre Pokémon do mundo! Ash Ketchum nos ensinou sobre perseverança e alcançar nossos sonhos e objetivos, e nos mostrou também que os pokémons que ele capturava eram armazenados em um dispositivo criado pelo Professor Carvalho, chamado Pokédex.
 
Então significa que vocês montarão uma Pokédex? Não! Vocês montarão uma "WhateverDex"
 
O que é uma WhaateverDex? Um dispositivo de armazenamento, DO QUE VOCÊ QUISER! Você pode criar uma Dex inspirada nos monstrinhos do Ben 10, magias e potions de D&D, armamentos disponíveis em MapleStory, ou até mesmo a discografia de uma banda!
 
Embora os exemplos aqui sejam mais voltados a jogos, a criatividade está liberada! Mas existem algumas coisas que vocês precisam se atentar, que são obrigatórias:
 
- Precisa ter uma lista de cards

- A interface visual precisa ser organizada

- Precisa ter uma busca ou filtro

- Detalhes precisam poder ser visualizados
- Cada item precisa ter pelo menos: nome, imagem, descrição. Demais elementos devem ser orientados de acordo com a dex de vocês (por exemplo: se eu escolho fazer uma dex de batons, posso colocar como um elemento do card a marca do batom)
-MÍNIMO DE 10 ITENS
 
Utilizem o que vocês aprenderam sobre HTML, CSS, JS para Web e Bootstrap para realizar o desafio.
