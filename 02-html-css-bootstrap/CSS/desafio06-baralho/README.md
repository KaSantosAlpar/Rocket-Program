Desafio 6 
CSS 

Faça uma carta de Baralho! 
Usando apenas CSS e imagens de naipes de sua escolha, faça uma carta de baralho. 
A carta deve virar o lado ao passar o mouse! (procure na internet como fazer, caso tenha 
duvidas). 

A carta deve conter o número ou letra da carta nas posições corretas usando transform 
Os naipes podem ser imagens e devem ser posicionados corretamente usando transform 
Caso seja um valete, uma rainha ou um rei, deve conter a imagem no meio. 

Apenas a parte de trás da carta pode ser completamente uma imagem, a parte da frente 
deve ser uma composição.
