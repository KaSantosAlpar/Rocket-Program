CSS 

-> Organizem os documentos HTML de vocês: 
Agora vocês deverão organizar o conteúdo de vocês em pastas dessa forma: 

° index.html -> seu portifólio 
° assets/ -> pasta com o css, imagens e fontes (se tiver) do seu portifólio 
° cafeteria/ 
o index.html -> site da cafeteria, com a tabela de funcionamento e contato 
incluso 
o assets/ -> pasta com o css, imagens e fontes (se tiver) da cafeteria 
° seusite-table/ -> o nome do seu site de tabela (adicione “-table” ao final) 
o index.html -> o seu site de tabela 
o assets/ -> pasta com o css, imagens e fontes (se tiver) do seu site em 
tabela 
° podcast/ -> o nome do seu podcast 
o index.html -> o seu podcast 
o assets/ -> pasta com o css, imagens e fontes (se tiver) do seu podcast 
° blog/ -> aqui ficará o site que será desenvolvido durante o curso! 
o index.html -> lembre-se que todo site começa com o index.html 
o assets/ -> pasta com o css, imagens e fontes 


Aplique o CSS aprendido, corrija erros e TESTE o seu site.
