Desafio 1 

Bootstrap 

Objetivo: Criar uma landing page simples com 3 seções (hero, serviços e rodapé). 
Requisitos: 

° Use o container para controlar o layout. 

° Organize as seções usando row e col. 

° Cada seção deve se adaptar a diferentes tamanhos de tela (col-sm, col-md, col
lg). 

° A seção “serviços” deve apresentar 3 opções. 

° O conteúdo da seção de “serviços” deve exibir 1, 2 ou 3 colunas dependendo do 
tamanho da tela.
