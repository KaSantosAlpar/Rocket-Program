Desafio 4

Bootstrap

Objetivo: Criar um formulário simples com nome, e-mail, assunto e mensagem.
Requisitos:

 Use form-control, form-group, form-label, form-check, form-select.

 Aplique feedback visual com is-invalid e is-valid.

 O campo de assunto deve ser um select.

 Adicione um botão com btn btn-primary.

 Mostre uma mensagem de sucesso no envio (capture o envio do formulário com
JS para mostrar a mensagem de sucesso).
